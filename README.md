# 게임개발총론 베이직 응용반 — 인터랙티브 학습 자료

강의에서 사용하는 인터랙티브 다이어그램 모음입니다.
GitHub Pages로 호스팅하여 노션 임베드에 활용합니다.

## 폴더 구조

```
repo/
├── shared/                        # 공용 스타일·스크립트
│   ├── css/
│   │   └── common.css
│   └── js/
│       └── common.js
├── session01/
│   ├── 00_intro/                    # 슈퍼클래스 → 역할 분리 애니메이션
│   ├── 01_framework_relations/      # GameMode·GameState·Controller·Pawn 관계도
│   ├── 02_lifecycle/                # 액터 라이프사이클 타임라인
│   └── 03_gameinstance_subsystem/   # 서브시스템 스코프 다이어그램
└── README.md
```

## 사용법

각 폴더의 `index.html`을 브라우저에서 열거나,
GitHub Pages 배포 후 노션에 임베드하세요.
