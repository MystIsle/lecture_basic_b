벡터 기본 개념

탭을 전환하며 벡터의 세 가지 핵심을 체험합니다

벡터 좌표
FVector(1.00, 1.00, 0)
크기 (길이)
1.41
방향 = 화살표가 가리키는 곳, 크기 = 화살표의 길이. 끝점을 드래그해서 벡터를 바꿔보세요.
곱셈 값 (스칼라) 1.0
코드
Forward * 1.0 = FVector(1.00, 0.00, 0) — 크기: 1.00
벡터에 숫자를 곱하면 방향은 그대로, 크기만 변합니다. 게임에서 속도를 곱하는 것과 같습니다.
정규화 = 크기를 1로
Vector.GetSafeNormal() // 방향만 남기고 크기를 1로
정규화는 방향만 남기고 크기를 1로 만드는 과정입니다. GetForwardVector()는 이미 정규화된 벡터를 반환합니다.
